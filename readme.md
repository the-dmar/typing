# Typing Test

<p style="font-size: 1.2rem;">View a live demo <a href="https://stalwart-pie-e6ab76.netlify.app/test">here</a></p>
<img src="https://raw.githubusercontent.com/the-dmar/typing/master/assets/Typing.gif" />
<p style="font-size: 1.2rem;">I wasn't happy with the existing options to learn typing, so I decided to create my own! I learned quite a lot about typescript, nextjs, and have improved my typing speed significantly since the start of this project. I have quite a few features I'd like to add to the practice page, and i'm open to any and all feedback!</p>

### Built With

<p style="font-size: 1.2rem;">This project was built using NextJs, Typescript, and StyledComponents</p>

## Features

<ul>
    <li style="font-size: 1.2rem;">Results include words per minute, words typed, and characters typed</li>
    <li style="font-size: 1.2rem;">One, three, five, and ten minute testing intervals</li>
    <li style="font-size: 1.2rem;">Dynamic text using a random paragraph API</li>
    <li style="font-size: 1.2rem;">Correct, complete, and incomplete letters indicated using conditional styles</li>
    <li style="font-size: 1.2rem;">useTypingText - Custom hook which iterates through paragraphs split by new line to display them to the user one at a time</li>
    <li style="font-size: 1.2rem;">useTimer - Custom hook to start, and reset a timer</li>
</ul>
